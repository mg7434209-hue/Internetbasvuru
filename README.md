# Internetbasvuru.com

Türk Telekom Yetkili Bayi - İnternet Başvuru Sitesi

## Mimari

- **Frontend**: Next.js 15 + React 18 (Railway)
- **Backend**: PHP + MySQL (Turhost)
- **DNS/CDN**: Cloudflare

## Geliştirme

```bash
npm install
npm run dev
```

## Deploy

Railway'e otomatik deploy: `git push` yeterli.

## Ortam Değişkenleri

`.env.example` dosyasını referans al, Railway'de Variables sekmesinden ekle.
